# NOTES
## 输入输出
1. %xx.xf 默认右对齐,  %-xx.xf 左对齐
2. %.xg 保留x位`有效数字`,科学计数法输出(%.xe 保留x位`小数`,科学计数法)
3. **scanf读取double要加上“l”（小写L）修饰符！！！！！！！！！！！！！**
4. pow的底数只能为正数
5. 开立方（包括正负）可以用cbrt（）
## 头文件包含关系
1. pow()在<math.h>
2. <ctype.h>:toupper()转大写, tolower转小写
