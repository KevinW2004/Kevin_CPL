## 基本思路
1. 使用字符界面, 利用sleep（）以及重复清屏加输出数组(其实就是屏幕画面)达到动态效果, 以画面左上角为[0,0]原点
2. 通过互联网获取windows下的改变光标位置和隐藏光标函数，做到不需要每次打印整个画面。
3. 所有的实体（恐龙、障碍）都是结构体，通过判断结构体中的xy是否重叠来判断碰撞
